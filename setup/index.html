Hermeer

Setup guides for Desktop and CLI modes

These guides walk you through connecting Hermeer to a depth companion on your computer. If you've never opened Terminal or PowerShell before, that's completely fine — every step is explained.

Desktop Mac or Windows · ~15 min CLI Mac or Windows · ~30 min

Desktop Mode

Desktop mode connects Claude Desktop (Anthropic's computer app) to your Hermeer data. Claude reads your entries, pulls, and threads — then writes interpretations back. They appear in the Hermeer app automatically.

What you'll need

  • A Mac or Windows computer
  • A Claude Pro or Max subscription from Anthropic
  • About 15 minutes
Step 1

Open your command line

You'll need to type a few commands during this setup. Don't worry — you'll just be copying and pasting, not writing anything from scratch.

Mac
Windows

On Mac, the app is called Terminal.

How to find it: Press Command + Space (a search bar appears at the top of your screen). Type Terminal and press Enter.

A window will open with a blinking cursor. That's Terminal. You'll paste commands here.

On Windows, the app is called PowerShell.

How to find it: Press the Windows key on your keyboard. Type PowerShell and click Windows PowerShell when it appears.

A blue window will open with a blinking cursor. That's PowerShell. You'll paste commands here.

Step 2

Install Node.js

Node.js is a free tool your computer needs to run the Hermeer connector. You only install it once.

Mac
Windows

Paste this into Terminal and press Enter:

brew install node

If you see "command not found: brew" — that's okay. You need to install Homebrew first. Paste this into Terminal, press Enter, and follow the prompts. Then try the command above again:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Alternatively, go to nodejs.org and download the Mac installer. Install it like any other app.

Go to nodejs.org and click the big green download button.

Run the installer that downloads. Click Next through each screen — the default options are all fine. Click Install at the end.

After installing: Close PowerShell and reopen it (so it can find Node.js).

Step 3

Install Git (if needed)

Git is a free tool that downloads the Hermeer connector. Your computer may already have it. Let's check.

Mac
Windows

Paste this into Terminal and press Enter:

git --version

If you see a version number (like "git version 2.39.0"), you're good — skip to Step 4.

If a popup appears asking to install "command line developer tools," click Install and wait for it to finish. This takes a few minutes.

Paste this into PowerShell and press Enter:

git --version

If you see a version number, you're good — skip to Step 4.

If you see an error, go to git-scm.com/downloads/win, download the installer, and run it. Use all the default options. Then close and reopen PowerShell.

Step 4

Install the Hermeer connector

This downloads a small piece of software that lets Claude Desktop talk to your Hermeer data.

Mac
Windows

Paste this into Terminal and press Enter:

git clone https://github.com/hermeerapp/hermeer-mcp-server.git ~/.hermeer-mcp-server && cd ~/.hermeer-mcp-server && npm install && npm run build

Paste this into PowerShell and press Enter:

git clone https://github.com/hermeerapp/hermeer-mcp-server.git $HOME\.hermeer-mcp-server; cd $HOME\.hermeer-mcp-server; npm install; npm run build

You'll see a bunch of text scroll by. When it finishes and you see the blinking cursor again, it worked.

Step 5

Download Claude Desktop

If you don't have it yet, download Claude Desktop from Anthropic.

Mac
Windows

Go to claude.ai/download. Download the Mac version and drag it into your Applications folder. Open it and sign in.

Go to claude.ai/download. Download the Windows version and run the installer. Open it and sign in.

Step 6

Connect the connector to Claude Desktop

Claude Desktop needs to know about the connector you just installed. You'll do this by running one command that creates a settings file.

Mac
Windows

Paste this into Terminal and press Enter:

mkdir -p ~/Library/Application\ Support/Claude && cat > ~/Library/Application\ Support/Claude/claude_desktop_config.json << 'EOF' { "mcpServers": { "hermeer": { "command": "node", "args": ["$HOME/.hermeer-mcp-server/dist/index.js"] } } } EOF

Now quit Claude Desktop completely (Cmd+Q) and reopen it.

Paste this into PowerShell and press Enter:

$configDir = "$env:APPDATA\Claude" New-Item -ItemType Directory -Force -Path $configDir | Out-Null @' { "mcpServers": { "hermeer": { "command": "node", "args": ["$HOME\\.hermeer-mcp-server\\dist\\index.js"] } } } '@ | Set-Content "$configDir\claude_desktop_config.json"

Now close Claude Desktop completely (right-click the icon in the taskbar and click Quit) and reopen it.

If you already have other MCP servers set up in Claude Desktop, you'll need to merge this into your existing config file rather than replacing it. Open the config file and add the "hermeer" entry inside "mcpServers".

Step 7

Set up your Depth Work project

In Claude Desktop:

  1. Click Projects in the sidebar
  2. Click Create Project
  3. Name it Depth Work
  4. Click the project, then click Set project instructions
  5. Paste the companion instructions from Hermeer

Getting the instructions from your phone to your computer: In the Hermeer app, go to Settings → Depth Companion → Copy Project Instructions. Then send them to yourself using one of these methods:

  • Email: Paste into an email to yourself, open it on your computer, copy from there
  • Notes: Paste into Apple Notes or Google Keep — it syncs to your computer automatically
  • AirDrop (Mac only): Paste into Notes on your phone, then share via AirDrop to your Mac
Step 8

Try it

Open a new chat inside your Depth Work project and say:

"Read my current state and tell me what's alive."

Claude will read your practice data through the connector and respond with a depth reading. Any interpretations it writes will appear in the Hermeer app on the entries they reference.

That's it. Desktop mode is set up. Claude Desktop can now read your entries, pulls, dreams, and threads — and write interpretations back to the app.

CLI Mode

CLI mode adds automation on top of Desktop mode — morning briefings, evening readings, and weekly syntheses that run on a schedule and appear in the app. It uses Claude Code, Anthropic's command-line tool.

Complete Desktop mode first. CLI mode builds on top of it.

What you'll need

  • Desktop mode already set up (see above)
  • A Mac or Windows computer
  • A Claude Max subscription (CLI mode uses Claude Code, which requires Max)
  • About 15 more minutes
Step A

Install Claude Code

Claude Code is a command-line version of Claude that can run automated tasks.

Mac
Windows

Paste this into Terminal and press Enter:

npm install -g @anthropic-ai/claude-code

When it finishes, type claude and press Enter to make sure it works. You should see Claude Code start up. Type /exit to close it.

Paste this into PowerShell and press Enter:

npm install -g @anthropic-ai/claude-code

When it finishes, type claude and press Enter to make sure it works. You should see Claude Code start up. Type /exit to close it.

Step B

Run the setup script

This script sets up your depth practice commands. It will ask you a few questions (your name, birth data, where your Hermeer sync folder is) and create everything automatically.

Mac
Windows

Paste this into Terminal and press Enter:

curl -sO https://raw.githubusercontent.com/hermeerapp/hermeer-mcp-server/main/scripts/setup-depth-practice.sh && bash setup-depth-practice.sh

Follow the prompts. When it finishes, close Terminal and reopen it so the new commands become available.

Paste this into PowerShell and press Enter:

Invoke-WebRequest -Uri "https://raw.githubusercontent.com/hermeerapp/hermeer-mcp-server/main/scripts/setup-depth-practice.ps1" -OutFile "setup-depth-practice.ps1"; powershell -ExecutionPolicy Bypass -File "setup-depth-practice.ps1"

Follow the prompts. When it finishes, close PowerShell and reopen it so the new commands become available.

Step C

Try your new commands

Mac
Windows

You now have two commands you can type in Terminal anytime:

You now have two commands you can type in PowerShell anytime:

  • morning — a morning briefing based on your practice, transits, and recent themes. On Sundays it automatically includes a weekly synthesis and forecast.
  • evening — an evening reflection and day-arc reading

Try typing morning and pressing Enter. Claude will read your data and write a briefing that appears in the Hermeer app.

That's it. CLI mode is set up. Your morning and evening commands are ready.

Troubleshooting

"command not found" errors

This usually means a tool wasn't installed properly, or you need to restart your command line. Close Terminal (Mac) or PowerShell (Windows) completely and reopen it. If the problem persists, go back to the step where you installed that tool and try again.

Interpretations aren't appearing in the app

Make sure your Hermeer sync folder is set up. In the Hermeer app: Settings → Sync. The connector looks for your data in the sync folder — if sync isn't configured, there's nothing to read.

Claude Desktop doesn't see the connector

After editing the config file (Step 6), you must completely quit Claude Desktop and reopen it. Just closing the window isn't enough — you need to fully quit the app.

Mac
Windows

Quit with Cmd+Q. To check the config file, paste this in Terminal:

cat ~/Library/Application\ Support/Claude/claude_desktop_config.json

You should see the "hermeer" entry. If the file is empty or missing, go back to Step 6.

Right-click the Claude icon in the taskbar and click Quit. To check the config file, paste this in PowerShell:

Get-Content "$env:APPDATA\Claude\claude_desktop_config.json"

You should see the "hermeer" entry. If the file is empty or missing, go back to Step 6.
